About Me

As a university student and a graduate student, I was interested in the security field, so I got the 'CISSP' certificate, and I was especially interested in 'encryption' among the security fields.
When I was a graduate student, 'academic supervisor' got an Android project, and I became interested in Android, so I started to step into society for the first time.
In the early days of Android, I joined as a mobile app developer.
Before the 'Bio healthcare' field attracted attention, I was convinced that I could join a venture company and lead the market, but the reality was not.
I was tired of 'politics in the company' and had a personal time for a while.
Currently, I operate 'Git Server', 'Multimedia Streaming Server', and 'File Server' personally, and I am working on a project that integrates 'AI', 'Robot', and 'Bio data'.
I can do 'Server management and programming for connection another device', 'PC program development', 'Android app development', 'PCB design and production', 'FPGA design and production', 'Firmware creation (prefer STM series)', '3D Design', '3D Printer Furnace printing and processing ',' Algorithm creation ',' Data analysis', 'Use of multimedia programs (video, photos)', etc.

Projects

Robot with bio signal

The goal of this robot arm is to be driven by EMG.
So far, it can be controlled by the IMU. IMU module, control module, and 3D Design was also made by me.
This robot arm can be very fast working, and the type of motor is being changed for testing, because I am trying to be reduce movement sound and to be more smooth movement.

ECG System

This project is based on a chip developed by a company that worked in the past.
It supports a small size of 1 channel or 10 channels, and aimed to support BLE.
We have written a program to control the BLE code and the developed chipset, and we have also created a mobile program that can be linked with the system.

FPGA Design

This is an FPGA designed to integrate AI systems with chipsets from companies that worked in the past.
I used Artix-7, and there is not much additional work due to different project schedule during the project.
It started for the first time in Vivado 2014.4, and even tested the chipset developed using Custom IP and internal interface.

Bio Chip test

In order to test the developed bio chipset, we directly made a control board and a chipset board.
The basic operation is the basic function of controlling and graphic expression with the mobile phone by making a Bluetooth connection with the mobile phone.

Set up for IC Design system

It is a program for semiconductor design, and it is set directly on the Linux system.
It worked on Redhat, Ubuntu, and Linux Mint.
I set up each on my laptop and workstation, and on some other devices on additional requests.

Blood Pressure Monitoring System

This is a project in which a paper is listed in the 'Scientific Report'.
Pneumatic sensors were used, and the first goal was to compare the extracted data by connecting to the blood pressure monitor that was previously sold.
This is a project where Korean 'YTN Science' and various articles were registered.

Sleep and walk pattern system

This project was conducted before the 'Behavior Pattern Analysis Program' became popular.

Basic functions are analyzing sleep patterns and calculating calorie consumption according to movement.

Chip Test Program
